# JobSphere 🚀

A modern **full-stack job portal** built with Spring Boot + React + TypeScript. Connect recruiters with job seekers in a clean, professional environment.

## ✨ Features

### For Job Seekers
- Browse and search jobs with smart filtering (keyword + location)
- Apply to jobs with resume upload (PDF)
- Track all applications with status updates
- View resume links

### For Recruiters
- Post new job listings
- View all received applications in real-time
- Accept / Reject candidates
- Dashboard with pagination and sorting

### General
- Secure JWT Authentication + Role-based access
- Responsive dark modern UI
- File upload handling (resumes)
- Clean pagination on job listings

## 🛠️ Tech Stack

### Backend
- **Spring Boot 3** + **Spring Security**
- **JPA / Hibernate** + **MySQL**
- **JWT Authentication**
- **Lombok** + **Spring Data JPA**

### Frontend
- **React 18** + **TypeScript**
- **Vite** (Build Tool)
- **React Bootstrap** + Custom CSS
- **Axios** + **React Router**

### Database
- **MySQL**
